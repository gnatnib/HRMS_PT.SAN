# Technology Stack

## Backend Technologies

### Core Framework
- **Laravel 10.x** - Main PHP framework
- **PHP 8.1+** - Required PHP version

### Authentication & Authorization
- **Laravel Fortify** - Authentication scaffolding
- **Laravel Sanctum** - API authentication
- **Spatie Laravel Permission** - Role-based permissions

### Database & ORM
- **MySQL** - Primary database
- **Eloquent ORM** - Laravel's built-in ORM
- **Database Migrations** - Schema version control

### Frontend Technologies

### JavaScript Framework
- **React 19.2.3** - Modern React with latest features
- **Inertia.js** - SPA-like experience without API
- **Vite 6.4.1** - Build tool and dev server

### UI Components & Styling
- **Tailwind CSS 3.4.17** - Utility-first CSS framework
- **Headless UI React 2.2.0** - Unstyled accessible components
- **Heroicons React 2.2.0** - SVG icon library
